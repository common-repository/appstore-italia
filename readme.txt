AppStore Italy is a plugin that allows you to include URL, price, photos, vote, category of the iphone/ipad application in your page.

== Description ==

Use this plugin if you want to easily add a link to your page with the loaded applications on the AppStore for iPhone and iPad. 

Will be shown detailed information such as application name, category, price, rating, links for download.

To add the link you just use [app #idnumber#] in your article, pages or comments. The ID number is the number from the official AppStore URL.

ITA:
